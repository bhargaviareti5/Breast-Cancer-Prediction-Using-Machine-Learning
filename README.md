# Breast Cancer Prediction Using Machine Learning

Breast Cancer Prediction Using Machine Learning is a web application developed using Python, Django, and Machine Learning. The application predicts whether a breast tumor is Benign or Malignant based on user input.

## Features

- User Registration
- User Login
- Breast Cancer Prediction
- View Dataset
- Machine Learning Integration
- User-Friendly Interface

## Technologies Used

- Python
- Django
- Machine Learning
- HTML
- CSS
- JavaScript
- SQLite

## Installation

1. Clone the repository

```bash
git clone https://github.com/bhargaviareti5/Breast-Cancer-Prediction-Using-Machine-Learning.git
```

2. Move to the project folder

```bash
cd Breast-Cancer-Prediction-Using-Machine-Learning
```

3. Install Django

```bash
pip install django
```

4. Run the server

```bash
python manage.py runserver
```

5. Open in your browser

```
http://127.0.0.1:8000/
```

## Usage

1. Register a new account.
2. Login to your account.
3. Open the Prediction page.
4. Enter the required details.
5. Click **Predict**.
6. View the prediction result.
